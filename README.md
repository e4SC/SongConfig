# SongConfig 구성
## 노래 추가
- 노래의 정답, 힌트, 2차힌트, 문제 등 정보를 추가합니다.
 <img width="1400" height="1021" alt="Image" src="https://github.com/user-attachments/assets/bb9e0b4a-5f48-447e-b579-1ec411db1b3e" />

 #### - 애니 / 작곡가 - 주제를 적으시면 됩니다. (X) : (Y) 이런식으로 출제가되면 이 부분에 적히는게 X에 해당하는 값입니다.
   
 #### - 제목 정답~힌트 - 주제의 정답과 힌트를 적으시면 됩니다. 여기 정답 부분에 적히는게 Y에 해당하는 값입니다.
   
${\textsf{\color{blue}Ex) 주제 - 애니, 제목정답 체인소맨 > 애니 : 체인소맨 이렇게 출력이 됩니다.}}$

 #### - 노래 정답~힌트 - 노래의 정답과 힌트를 적으시면 됩니다.
   
 #### - 가수1-7 정답~힌트 - 가수의 정답과 힌트를 적으시면 됩니다.

### ${\textsf{\color{red}주제와 노래는 반드시 작성하셔야 합니다.}}$

### ${\textsf{\color{red}가수는 최대 7명까지 작성 가능하며, 최소 1명의 가수는 적으셔야 합니다.}}$

### ${\textsf{\color{red}가수는 연속적으로 작성하셔야 합니다. 가수가 2명이면 가수1, 가수2를 작성하셔야합니다.}}$

### ${\textsf{\color{red}가수1, 가수3 이렇게 작성하시면 안됩니다.}}$

## 노래 수정
- 추가된 노래의 정보를 불러와서 수정 및 제거. 노래끼리의 순서 변경이 가능합니다.
<img width="385" height="292" alt="Image" src="https://github.com/user-attachments/assets/d74914ea-851f-46e3-be95-c47c72b5ef36" />

- 처음 추가한 노래가 인덱스 1번이 됩니다. 5번째로 추가한 노래는 인덱스 5번이 됩니다.
- 수정하고 싶은 노래의 인덱스 번호를 입력하고 노래 수정을 하면 추가해둔 노래의 정보를 불러옵니다.
- 노래 리스트에서 추가된 모든 노래의 인덱스를 한 번에 확인할 수 있으며 더블 클릭으로 바로 수정이 됩니다.
<img width="435" height="493" alt="Image" src="https://github.com/user-attachments/assets/70dbef88-d451-4e54-8d48-3c5444d22b69" />

- 노래 순서 변경은 노래가 5개 있다고 가정 했을 때 1번과 3번의 순서를 바꾸고 싶으면

  - 왼쪽에 1, 오른쪽에 3을 적고 노래순서변경을 누르시면 됩니다.

  - 그럼 1번노래 - 안녕하세요, 3번노래 - 반갑습니다 일 경우

  - 1번노래 - 반갑습니다, 3번노래 - 안녕하세요 로 순서 변경이 됩니다.
    

- 노래 제거는 노래의 인덱스 번호를 적고 제거를 하시면 해당 노래의 정보가 삭제됩니다.
- 모두 제거는 저장된 노래의 모든 정보를 한 번에 제거 합니다.
  

## 노래 백업
- 추가한 노래들을 백업파일을 만들어 백업합니다.

## 장르 설정
- 노래의 장르와 곡 수를 설정합니다.

- 배포한 소스에서는 장르 선택기능을 비활성화 했기 때문에 장르 갯수 1 설정 > 장르이름을 아무거나 정해주시고 곡수를 추가한 곡수로 설정해주세요.

- 장르 갯수는 1개여야 합니다. 장르 선택 기능을 추가하고 싶으면 따로 연락주세요.

## 맵 설정
- 스피드모드 힌트 출력시간, 감상모드 힌트 출력시간

- 스킵설정, 음악 셔플, 주관식, 객관식 문제 사용 여부를 설정할 수 있습니다.

## 음원자르기 / 믹스
<img width="306" height="392" alt="Image" src="https://github.com/user-attachments/assets/32978117-d729-40bf-8e32-d7db99118ede" />

### 자르기
- mp3, ogg, wav 파일을 지원합니다.
- 엑셀파일의 n행 ~ n행까지 자를 범위를 지정할 수 있습니다.

- 엑셀 파일이 같이 실행되며 Cut 시트에 형식대로 작성 후 저장을 하고 나서 프로그램 설정 후 자르기를 하시면 sound 폴더에 잘린 노래가 생성됩니다.
### 믹스
- 최소 2개의 노래를 지정해야합니다. 최대 3개의 노래까지 섞을 수 있습니다.
- 각 노래의 자르고 싶은 시간을 지정 후 믹스 하면 자른 노래들을 합쳐줍니다.

- 믹스도 마찬가지로 Mix 시트에 형식대로 작성 후 저장을 하고 나서 프로그램 설정 후 믹스를 하시면 sound 폴더에 믹스된 노래가 생성됩니다.

# 음원 자르기 전 설정 사항
## 1. python 다운
https://www.python.org/ftp/python/3.10.11/python-3.10.11-amd64.exe
​
- 중간에 path 한번 뜰텐데 무조건 체크하기
- 나머진 기본 값
- path를 체크 못했을 땐 환경변수가서 설정 따로 해줘야하니 구글링해서 넣어주던가 python 자체를 삭제하고 다시 다운 받기를 권장합니다.
  
## 2. 1번을 수행하고 cmd에서 해당 명령어 입력

- 1) pip install yt_dlp
- 2) pip install pydub
- 3) pip install pandas
- 4) pip install openpyxl

## 3. 자르기/믹스 실행 (프로그램 기본값으로 자르시는게 가장 좋습니다.)
- SongConfig 실행 후 음원 자르기 버튼 클릭.
- 엑셀 실행이되면 Cut 또는 Mix 시트에서 형식에 맞춰서 작성.
- SongConfig로 돌아와서 mp3, ogg, wav 중 어떤 형식으로 저장할지 설정.
- 음량, n행 노래부터 n행 노래까지 자를지 설정.
- 자르기 또는 믹스 버튼 클릭.

### ※ 완료메세지가 출력되기 전까지 SongConfig와 엑셀파일은 만지지마세요. ※
### - 자르기 또는 믹스 중 취소를 누르게 되면 진행중이던 작업이 완료 된 후 창이 닫힙니다.
#### - 창이 그래도 안닫히면 작업관리자로 종료해주세요.

- 잘린 음원들은 sound\wav 혹은 sound\mp3 혹은 sound\ogg 혹은 sound\mix 파일에 저장됩니다.
